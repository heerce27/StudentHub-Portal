# StudentHub Portal

## Project Overview

StudentHub is a web-based student portal developed using HTML, CSS, PHP, and MySQL. The portal helps students manage academic information through a simple and responsive interface.

---

## Features

- Student Login
- Dashboard
- Student Profile
- Attendance
- Results
- Certificates
- Fee Details
- Events
- Feedback
- Contact Page

## Installation

1. Install XAMPP.
2. Copy the project folder into:
   ```
   htdocs/
   ```
3. Start Apache and MySQL.
4. Import the database:
   ```
   studenthub.sql
   ```
5. Open your browser and visit:
   ```
   http://localhost/StudentHub
   ```
